# role-ui-triage seed corpus

Eight manually-triaged findings from the operator sessions that produced
ADR-0061. These are the bootstrap labels for the first optimizer run.

Labels (`label_*` fields) reflect the operator's actual decisions at the
time of triage. A future CLI loader (deferred to Step 6) imports this
file into `triage_findings`.

- `label_is_real_bug`: true when the operator dispatched a fix or confirmed
  the defect; false when suppressed or escalated as infrastructure.
- `label_dispatch_action`: the action the operator would have taken had the
  schema been in place at the time.
- `label_severity`: operator's severity assessment (may differ from the
  role's initial classification).
- `label_notes`: free-text rationale for the label decision.
